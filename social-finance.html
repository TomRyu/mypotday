SECTION 01

핵심 모델 — "10만원으로 1,000만원" 마이크로렌딩 계

순수 계 구조를 넘어 플랫폼 선지급 마이크로렌딩으로 설계하면, 서민이 즉시 목돈을 받고 소액으로 분할 상환하는 강력한 금융 상품이 됩니다.

CORE MODEL — 플랫폼 선지급 구조
[ 가입 · 그룹 결성 ]
  지인 5~10명이 계 그룹 결성 (연대보증 구조)
  실명인증 + 소득 확인 + 플랫폼 심사 (자동화)

[ 즉시 지급 ]
  심사 통과 즉시 → 플랫폼이 1,000만원 선지급
  (기다릴 필요 없음 — 전통 계와 가장 큰 차이)

[ 분할 상환 ]
  수령자: 매월 10만원 × 100개월 (8.3년) 자동이체 상환
  그룹원: 연대 압박 역할 → 이탈 시 그룹 전체 불이익

[ 다음 순번 ]
  상환 실적 6개월 이상 → 다음 그룹원 순번 지급
  → 그룹 내 순환 반복

[ 수익 구조 ]
  플랫폼 수수료: 월 상환액의 2~3% (월 2,000~3,000원)
  또는 대출 이자: 연 3~6% (서민금융진흥원 기준 금리)
서민 입장에서의 임팩트: 저축은행 대출(연 15%)과 비교 시 1,000만원 기준 연간 이자 절감액 약 90만원. 사채(연 30%) 대비 연간 240만원 절감. 금융 비용 절감 = 실질 소득 증가입니다.

SECTION 02

서민금융진흥원 파트너십 — 구체화 방안

서민금융진흥원은 MyPotDay가 필요한 두 가지 — 자금과 신뢰(라이선스 대체) — 를 동시에 제공할 수 있는 핵심 파트너입니다.

2-1 · 서민금융진흥원 현황
항목내용
연간 운용 자금2025년 기준 11~12조원 (역대 최대 규모)
주요 상품햇살론(근로자·유스·뱅크), 미소금융, 햇살론15(최저신용자)
미소금융 조건한도 최대 7,000만원 · 금리 연 2~4.5% · 무담보·무보증
햇살론 조건한도 최대 2,000만원 · 대상: 연소득 3,500만원 이하 또는 신용평점 하위 20%
협약기관 채널kinfa.or.kr → 협약사업 신청 / 서민금융콜센터 1397
2-2 · MyPotDay가 플랫폼 역할을 하는 구체적 방안
01
협약기관 등록
MyPotDay가 서민금융진흥원의 공식 협약 플랫폼으로 등록. 서민금융진흥원이 자금을 제공하고, MyPotDay가 모집·심사·배분·상환 관리를 전담하는 구조. 별도 금융 라이선스 없이 시작 가능.
02
디지털 심사 자동화
서민금융진흥원의 기존 심사 기준(소득 확인, 신용평점)을 플랫폼에 내재화. 신청 → 서류 업로드 → AI 사전 심사 → 최종 확인의 전 과정을 앱 내에서 처리. 기존 오프라인 창구 방문 불필요.
03
그룹 연대보증 시스템 운영
지인 5~10명 그룹을 계 단위로 묶어 연대보증 구조를 디지털화. 그룹 내 한 명 연체 시 → 전체 그룹에 알림 발송 + 다음 순번 지급 보류. 사회적 압력이 가장 강력한 상환 메커니즘.
04
상환 모니터링 & 데이터 피드백
상환율·연체율·그룹별 리스크 데이터를 실시간으로 서민금융진흥원에 제공. 플랫폼이 축적한 '계 신용 데이터'는 기존 금융권 신용평가로 잡히지 않는 서민의 금융 이력을 최초로 데이터화.
05
성과 기반 자금 확대
초기 파일럿(100계 그룹) 상환율 검증 → 데이터 기반 자금 증액 협의. 상환율 95% 이상 달성 시 미소금융 한도 확대 및 독자 라이선스 취득 근거 자료로 활용.
핵심 가치 교환: 서민금융진흥원 → MyPotDay에 자금 + 신뢰 제공. MyPotDay → 서민금융진흥원에 디지털 채널 + 데이터 + 상환율 개선 제공. Win-Win 구조로 진입 장벽이 낮습니다.
2-3 · 파트너십 진입 전략
PARTNERSHIP ENTRY STRATEGY
Step 1 (창업 직후)
  → 서민금융진흥원 협약사업팀 접촉 (kinfa.or.kr → 협약기관 신청)
  → "디지털 계 기반 마이크로파이낸스 플랫폼" 제안서 제출
  → 금융규제 샌드박스 혁신금융서비스 동시 신청

Step 2 (파일럿 협의)
  → 소규모 파일럿 제안: 100계 그룹, 총 대출 10억원 규모
  → 미소금융 자금으로 파일럿 실행 (무담보·무보증 조건 활용)
  → 6개월 상환 데이터 확보

Step 3 (확대)
  → 파일럿 상환율 95%+ 달성 시 자금 규모 확대 협의
  → 햇살론 연계 상품 추가 설계
  → 언론 보도 → 임팩트 투자 유치 근거

예상 소요 시간: 협약 체결까지 3~6개월
SECTION 03

한국판 그라민 은행 포지셔닝 — 심층 검토

그라민 은행은 1983년 방글라데시에서 시작해 노벨 평화상을 받은 마이크로파이낸스 기관입니다. MyPotDay가 이 모델을 한국에서 디지털로 재현할 수 있는지 구체적으로 검토합니다.

요소그라민 은행MyPotDay 버전

상환 방식 주(週)단위 소액 분할 상환 월(月)단위 자동이체 상환

SECTION 04

규제 전략 — 단계별 라이선스 로드맵

0단계
서민금융진흥원 협약기관 등록 (라이선스 불필요)
독자 금융 라이선스 없이 서민금융진흥원의 협약 플랫폼으로 운영 시작. 자금은 서민금융진흥원에서, 심사·배분·상환 관리는 MyPotDay가 담당. 자본금 요건 없음. 가장 빠른 진입 경로.
1단계
혁신금융서비스 (샌드박스) 신청
금융위원회 혁신금융서비스 지정 신청. 코나아이 선례 활용. 지정 시 최대 4년간 규제 유예 + 정부 신뢰 확보. 투자 유치 시 가장 강력한 신뢰 신호.
2단계
대부업 등록 (소규모 독자 운용 시)
자체 자금으로 소규모 대출 운용 시 필요. 자본금 1,000만원, 지자체 등록. 대부업 교육 수료. 규모가 작을 때 가장 빠르고 저렴한 독자 라이선스.
3단계
온라인투자연계금융업 (온투업) 등록
P2P 금융 플랫폼으로 성장 시 필요. 최소 자기자본 5억원 (연계대출 300억 미만 기준). 금융위원회 등록. MAU 1만 이후 추진.
4단계
전자금융업 등록 (에스크로 자체 운용)
플랫폼이 직접 에스크로를 운용하고 자금 이동을 담당할 때 필요. MAU 5만 이후 중기 목표.
SECTION 05

실행 로드맵

즉시 · 창업 직후
서민금융진흥원 협약 접촉 + 혁신금융 샌드박스 신청 kinfa.or.kr 협약사업 신청서 제출. 제안서 핵심: "디지털 계 기반 마이크로파이낸스 플랫폼, 연대보증 그룹으로 상환율 95%+ 목표". 금융규제 샌드박스 동시 신청.
Phase 1 · 0~6개월
파일럿 100계 그룹 운영 (총 대출 1~10억원) 지인 기반 소규모 그룹으로 시작. 서민금융진흥원 미소금융 자금 연계. 상환 데이터 6개월치 확보. 임팩트 투자자 대상 데모데이 참가.
Phase 2 · 6~18개월
정식 앱 출시 + 자금 규모 확대 파일럿 상환율 데이터 기반 서민금융진흥원 자금 확대 협의. 햇살론 연계 상품 추가. 임팩트 VC Pre-Seed 3~5억원 유치 목표. MAU 1만 달성.
Phase 3 · 2~3년
온투업 라이선스 취득 + 독자 자금 운용 온라인투자연계금융업 등록. 계 신용 데이터 → 중금리 대출 상품 설계. B2B(소상공인 단체, 신협, 새마을금고 제휴). MAU 10만 목표.
Phase 4 · 3~5년
K-마이크로파이낸스 글로벌 수출 동남아(ROSCA 문화권: 인도네시아 아리산, 필리핀 팔루완), 아프리카(케냐·나이지리아) 진출. 한국판 그라민 은행 모델의 글로벌 라이선싱.
