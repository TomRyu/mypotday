곗돈 타는 날 — 경쟁사분석 사업요약

계모아 (GyeMoa)

디지털 계(契) 플랫폼 — 경쟁사 현황 및 사업계획 요약

작성일: 2026년 5월

1. 경쟁사 현황 분석

결론부터 — 시장은 열려 있습니다. 전문 디지털 계 서비스가 거의 없고, 있는 것도 기능이 제한적입니다.

1-1. 직접 경쟁사

① 코나아이 (계모임 플랫폼)

코나아이가 선불전자지급 수단 기반으로 지인 간 계모임을 지원하는 플랫폼을 혁신금융서비스로 지정받아 운영 중입니다. 단, B2C 전용 앱으로 대중화되지 않고 제한적 서비스에 그쳤습니다.

② 토스 / 카카오뱅크 — 모임통장

계(契) 전용은 아니지만 모임통장 기능으로 일부 수요를 흡수 중입니다. 그러나 자동 순번 배분·곗돈 관리 기능이 없어 진정한 의미의 계 앱이 아닙니다.

1-2. 직접 경쟁사 기능 비교

비교 항목 토스 모임통장 카카오뱅크 코나아이 계모아
자동 순번 배분 ❌ ❌ 일부 ✅ 완전 자동
미납 자동 독촉 ❌ ❌ ❌ ✅
에스크로 보관 ❌ ❌ 부분 ✅
목적별 템플릿 ❌ ❌ ❌ ✅ 5종 제공
전용 앱 슈퍼앱 내 기능 슈퍼앱 내 기능 제한적 ✅ 전문 앱

→ 직접 경쟁사 사실상 없음. 완전한 블루오션 시장.

1-3. 간접 경쟁사

서비스 성격 한계
토스 모임통장 공동 잔액 관리 순번계 기능 없음
카카오뱅크 모임통장 공동 잔액 관리 순번계 기능 없음
네이버페이 더치페이 1회성 정산 지속 계모임 불가
엑셀·카카오톡 수동 관리 여전히 대부분 이렇게 함

아직도 대부분 카톡 + 계좌이체 + 엑셀로 수동 관리합니다. 이것이 계모아의 핵심 기회입니다.

1-4. 시장 기회 포인트

  • 자동 순번 추첨·배정 (찬반 투표 포함)

  • 납입 독촉 자동화 (미납자 알림)

  • 계주 없는 투명한 공동 관리 (불신 해소)

  • 목적별 계 템플릿 (여행계·결혼계·친목계)

  • 연체자 신고·제명 기능

  • 글로벌 확장 여지 — 동남아·아프리카 Chama·Arisan 문화권

2. 시장 분석

2-1. 시장 규모

구분 규모 비고
계모임 경험 성인 인구 약 2,500만 명 이상 성인의 약 50%
연간 계 거래 추정액 약 30~50조원 비공식 추정치
디지털 전환 TAM 약 1~3조원 수수료 0.5~1% 기준
3년차 목표 SOM 약 150억원 거래액 1,500억 × 1%

2-2. 타깃 고객

◆ 1차 타깃 — 계주 (모임 주도자)

  • 30~50대 여성 (전통적 계모임 주도 계층)

  • 매달 납입 독촉·정산에 지쳐있는 계주

  • 월 평균 10~20명 규모 계모임 운영

◆ 2차 타깃 — 계원 (참여자)

  • 목돈 마련이 필요한 MZ세대 직장인

  • 기존 계모임에서 불신·분쟁 경험이 있는 사용자

  • 안전한 온라인 계모임을 원하는 비대면 선호 세대

3. 서비스 핵심 기능

① 계 개설 및 초대

  • 계 유형: 순번계, 추첨계, 적금계, 목표계

  • 목적별 템플릿: 여행계 / 결혼계 / 친목계 / 적금계 / 자유계

  • 카카오톡·문자·링크로 계원 초대 (비앱 사용자도 웹 참여)

② 납입 및 정산 자동화

  • 자동이체 연동: 납입일 D-3, D-1, D-day 알림 자동 발송

  • 미납 시 3단계 에스컬레이션 (알림 → 계원 공지 → 계주 알림)

  • 곗돈 수령자에게 에스크로 자동 이체

③ 투명한 자금 관리 (에스크로)

  • 납입금은 계모아 에스크로 계좌에 보관 (계주 임의 인출 불가)

  • 전체 계원이 잔액·납입 현황 실시간 열람

  • 계 종료 시 자동 정산 및 영수증 발급

④ 계원 신용도 시스템

  • 납입 이력 기반 '계 신용도' 부여

  • 연체·제명 시 신용도 하락 → 타 계 가입 제한

⑤ 프리미엄 기능

  • 계 보험 (먹튀·연체 보장 상품 — 보험사 제휴)

  • 계 역사 아카이브 (과거 계 이력 PDF 다운로드)

  • 다계좌 연동 (한 앱에서 여러 계 동시 관리)

5. 기술 스택

  • 모바일 앱: Flutter (iOS/Android 동시 지원)

  • 백엔드: Spring Boot (REST API)

  • 결제·에스크로: 포트원(구 아임포트) PG 연동 / 토스페이먼츠 에스크로 API

  • 인프라: AWS (EC2 + RDS + S3)

  • 알림: Firebase FCM (푸시) + Kakao 알림톡 (납입 독촉)

  • 인증: 카카오·네이버 소셜 로그인 + PASS 본인인증

6. 결론

계모아는 블루오션 시장에서 Flutter 기반 빠른 MVP 개발이 가능하며, 대흥정보의 개발 역량을 직접 활용할 수 있는 최적의 창업 아이템입니다. 전문 경쟁사가 없는 지금이 시장 선점의 최적 타이밍입니다.

  • 전문 경쟁사 사실상 없음 → 블루오션

  • Flutter/Spring Boot 기존 역량 완전 활용 가능

  • 금융규제 샌드박스(혁신금융서비스) 선례 존재 (코나아이)

  • 5년차 MAU 300만, 연 거래액 8,000억, 매출 84억+ 목표
