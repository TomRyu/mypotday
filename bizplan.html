계모아 (GyeMoa)

한국 전통 계(契) 문화의 디지털 혁신 — 투명하고 안전한 모바일 계모임 플랫폼

사업 유형 핀테크 / 모바일 앱 (Flutter)

작성일 2026년 5월

1. 사업 개요

1.1 서비스 소개

계모아(GyeMoa)는 한국 전통 '계(契)' 문화를 모바일 앱으로 구현한 핀테크 플랫폼입니다. 현재 대부분의 계모임은 카카오톡 + 계좌이체 + 엑셀로 수동 관리되고 있어 계주의 부담, 미납 추적의 어려움, 횡령·먹튀 등 신뢰 문제가 지속적으로 발생하고 있습니다.

계모아는 계원 초대·납입·배분·정산의 전 과정을 자동화하고 투명하게 관리하여, 계주에게는 관리 부담 제거를, 계원에게는 안전한 목돈 마련 수단을 제공합니다.

1.2 사업 비전 및 목표

비전: 대한민국 3,000만 계모임 참여자가 신뢰하는 디지털 계 플랫폼 — 계모아
6개월 1년차 2년차 3년차
MVP 출시 베타 유저 1,000명 MAU 5만 명 계 거래액 50억 MAU 20만 명 계 거래액 300억 MAU 80만 명 계 거래액 1,500억

1.3 핵심 가치 제안

  • 계주 부담 제로 — 납입 독촉·정산·순번 배정 100% 자동화

  • 투명한 자금 흐름 — 모든 계원이 실시간 납입 현황 확인 가능

  • 안전한 보관 — 계좌 에스크로 방식으로 계주 임의 인출 불가

  • 목적별 계 템플릿 — 여행계·결혼계·친목계·적금계 원클릭 생성

  • 미납 자동 처리 — 연체자 알림·제명·대체 납입 플로우 자동화

2. 시장 분석

2.1 시장 규모

한국의 계모임 문화는 수백 년의 역사를 가진 고유한 소셜 금융 형태로, 현재도 광범위하게 활용되고 있습니다. 통계청 조사에 따르면 성인 2명 중 1명이 계모임 경험이 있으며, 연간 계 거래 추정액은 수십조 원에 달합니다.

구분 규모 비고
계모임 경험 성인 인구 약 2,500만 명 이상 성인의 약 50%
연간 계 거래 추정액 약 30~50조원 비공식 추정치
디지털 전환 TAM 약 1~3조원 수수료 0.5~1% 기준
당사 3년차 목표 SOM 약 150억원 거래액 1,500억 × 1%

2.2 경쟁사 현황

국내 디지털 계 전문 앱은 사실상 공백 상태입니다. 유사 서비스들은 모두 핵심 기능이 부재합니다.

비교 항목 토스 모임통장 카카오뱅크 코나아이 계모아
자동 순번 배분 ❌ ❌ 일부 ✅ 완전 자동
미납 자동 독촉 ❌ ❌ ❌ ✅
에스크로 보관 ❌ ❌ 부분 ✅
목적별 템플릿 ❌ ❌ ❌ ✅ 5종 제공
전용 앱 (Flutter) 슈퍼앱 내 기능 슈퍼앱 내 기능 제한적 ✅ 전문 앱

→ 직접 경쟁사 사실상 없음. 완전한 블루오션 시장.

2.3 타깃 고객

◆ 1차 타깃 — 계주 (모임 주도자)

  • 30~50대 여성 (전통적 계모임 주도 계층)

  • 매달 납입 독촉·정산에 지쳐있는 계주

  • 월 평균 10~20명 규모 계모임 운영

◆ 2차 타깃 — 계원 (참여자)

  • 목돈 마련이 필요한 MZ세대 직장인

  • 기존 계모임에서 불신·분쟁 경험이 있는 사용자

  • 안전한 온라인 계모임을 원하는 비대면 선호 세대

3. 서비스 기획

3.1 핵심 기능

◆ ① 계 개설 및 초대

  • 계 유형 선택: 순번계, 추첨계, 적금계, 목표계

  • 목적별 템플릿: 여행계 / 결혼계 / 친목계 / 적금계 / 자유계

  • 카카오톡·문자·링크로 계원 초대 (비앱 사용자도 웹으로 참여)

  • 계 규칙 설정: 납입 주기, 금액, 순번 방식, 연체 패널티

◆ ② 납입 및 정산 자동화

  • 자동이체 연동: 납입일 D-3, D-1, D-day 알림 자동 발송

  • 미납 시 3단계 에스컬레이션 (알림 → 계원 공지 → 계주 알림)

  • 납입 완료 시 전체 계원에게 실시간 투명 공개

  • 곗돈 수령자에게 자동 이체 (에스크로 → 수령자 계좌)

◆ ③ 투명한 자금 관리 (에스크로)

  • 납입금은 계모아 에스크로 계좌에 보관 (계주 임의 인출 불가)

  • 전체 계원이 잔액·납입 현황 실시간 열람

  • 계 종료 시 자동 정산 및 영수증 발급

◆ ④ 계원 관리

  • 계원 신용 점수 시스템 (납입 이력 기반 '계 신용도' 부여)

  • 연체·제명 시 계 신용도 하락 → 타 계 가입 제한

  • 계원 투표로 규칙 변경·제명 처리

◆ ⑤ 프리미엄 기능

  • 계 보험 (먹튀·연체 보장 상품 — 보험사 제휴)

  • 계 역사 아카이브 (과거 계 이력 PDF 다운로드)

  • 다계좌 연동 (한 앱에서 여러 계 동시 관리)

3.2 법적 구조

핵심: 전자금융업 등록 또는 기존 은행·PG사 제휴를 통해 에스크로 서비스 구현

  • 단기 전략: 기존 PG사(토스페이먼츠, 나이스페이 등) API 연동으로 에스크로 구현

  • 중기 전략: 전자금융업 등록 (소액후불결제업 또는 전자지급결제대행업)

  • 법률 자문: 금융규제 샌드박스 '혁신금융서비스' 신청 → 규제 유예 기간 확보

  • 참고: 코나아이가 동일 구조로 혁신금융서비스 지정 선례 있음
