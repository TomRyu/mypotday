01 · Core Decision

채팅 기능, 직접 만들어야 할까?

계원들 사이의 소통은 계모임 경험의 핵심입니다. 납입 일정 조율, 순번 협의, 한턱 날짜 잡기 — 다 대화가 필요합니다. 하지만 "채팅을 앱 안에 만든다"는 결정은 신중해야 합니다.

결론부터

채팅 기능은 직접 만들지 않습니다.
계원들은 이미 카카오톡 단체방을 씁니다.
그 행동을 앱으로 옮겨오려면 계원 전체가 앱에 가입해야 하는
마찰이 생깁니다. 채팅 설득하다 계 개설 자체를 잃을 수 있습니다.

MVP 전략: 카카오톡 연동 + 경량 공지 기능

02 · Three Options

소통 방식 3가지 옵션 비교

A

카카오톡 연동 MVP 추천

대화는 카톡에서, 납입·정산·알림은 앱에서 — 역할 분리

⭐ RECOMMENDED

작동 방식

계 개설 시 카카오톡 단체방 초대 링크를 자동 생성하여 계원들에게 발송. 대화는 카톡에서 진행하고, 앱은 납입·알림·정산 전용으로 역할 분리.

장점

  • 계원 전원 카톡 기사용 → 마찰 0
  • 개발 비용 없음
  • 카톡 UX에 익숙한 중장년층 친화
  • 알림톡 API로 앱→카톡 자동 공지 가능
  • 빠른 MVP 출시 가능

한계

  • 앱 이탈 후 카톡으로 전환되는 구조
  • 대화 내용이 앱에 기록 안 됨
  • 장기적으로 앱 체류 시간 감소 가능
B

경량 인앱 공지·반응 6개월 내 추가

채팅이 아닌 "공지 + 이모지 반응" — 운영에 필요한 소통만

PHASE 2

작동 방식

계주가 공지를 올리면 계원들이 이모지로 반응. 자유 대화 없이 운영에 필요한 소통만 커버. 납입 확인, 날짜 투표, 순번 당기기 신청 등 구조화된 소통.

기능 구성

  • 계주 공지 발행 (텍스트+이미지)
  • 계원 이모지 반응 (👍❤️🎉😢)
  • 날짜 선택 투표 (한턱 날짜 조율)
  • 순번 당기기 찬반 투표
  • 납입 완료 확인 체크

장점 / 한계

  • 개발 복잡도 낮음
  • 운영 소통은 100% 커버
  • 앱 내 기록 유지 가능
  • 자유 대화 불가 (카톡 병행 필요)
C

풀 인앱 채팅 MVP 비추천

실시간 채팅·사진·파일 공유 — 카카오톡 수준의 채팅방

PHASE 3+

작동 방식

앱 안에 계 전용 채팅방. 실시간 메시지·사진 공유·읽음 확인. 카카오톡을 완전히 대체하는 수준의 인앱 커뮤니케이션.

개발 난이도

  • WebSocket 실시간 서버 구축 필요
  • Firebase Realtime DB 또는 전용 서버
  • 푸시 알림 완전 재구현
  • 사진 저장·CDN 비용 추가
  • 개발 기간 +3~6개월, 비용 +수천만원

리스크

  • 카카오톡과 경쟁하는 구도
  • 계원 전원 앱 가입 강제 필요
  • 핵심 기능(계 운영) 개발 지연
  • 서버 유지비 급증

⚠ MVP에서 풀 채팅을 만들면 안 되는 이유

채팅은 네트워크 효과가 필요합니다. 계원 10명 중 8명이 앱을 설치해야 채팅이 의미 있는데, 그 가입 장벽을 넘는 게 계 서비스의 가장 어려운 부분입니다. 반면 카카오톡 링크를 공유하면 앱 미설치자도 단체방에 참여할 수 있습니다. 채팅 기능을 설득하다 계 개설 자체를 잃는 것이 MVP 단계의 가장 큰 리스크입니다.

03 · Kakao Integration

카카오톡 연동 상세 설계

계 개설 → 카카오톡 단체방 자동 연동 플로우

계주 계 개설
→
계원 초대 링크 생성
→
카카오톡 공유
→
단체방 자동 개설
→
앱 알림톡 연동
역할 분리 원칙: 카카오톡 = 자유로운 대화·친목·감성 소통 / 곗돈 타는 날 앱 = 납입 관리·알림·정산·공지·투표. 두 채널이 겹치지 않고 보완합니다.
기능채널설명
자유 대화·친목📱 카카오톡날씨 이야기, 계원 근황, 일상 대화
납입 독촉 알림🔔 알림톡 → 카카오톡D-3, D-1, D-Day 자동 발송
수령 축하 알림🔔 알림톡 → 카카오톡"🎉 오늘은 홍길동님의 곗돈 타는 날!" 전체 발송
공지·운영 안내📣 앱 공지판계주 공지, 계원 이모지 반응
날짜 투표🗳 앱 투표 기능한턱 날짜, 모임 일정 선택
순번 당기기 투표🗳 앱 투표 기능찬반 투표 → 자동 승인 처리
납입 내역 공유📊 앱 → 카카오톡"이번 달 납입 현황" 카드 자동 공유
한턱 쿠폰 발송🎁 앱 → 카카오톡선물 쿠폰을 카카오 선물하기로 전송
